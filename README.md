# IP-ku

> **Note**
>
> This is my practice project using Rust. There are many things I need to improve.

CLI to check IP address

## How to install

You must install [Cargo](https://doc.rust-lang.org/cargo/getting-started/installation.html).

```bash
cargo install ipku
```

## How to use

```bash
ipku
```
